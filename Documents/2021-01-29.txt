2)session:
-session save at server side.
-it can access in any page only including session_start();
-it can use in login.
-unset($_SESSION['username']); for unset perticular session.
-session_destroy() is use to destroy all session.

3)cookie:
-setcookie(key,value,time()+time to alive cookie)
-for unset cookie,that will be same :setcookie(key,value,time()-time to alive cookie)
-it will save at client end.
-it is use to save client activity values by website.
-like,use in amazon,flipcart.
