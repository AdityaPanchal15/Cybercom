Angular Fundamentals:
1)Building blocks of Angular Apps
-Components
	-encapsulate Data,HTML Template,Logic
-every component has root component.

2)steps:
	1)create component:
	    -name it courses.component.ts(corses component)
	2)register it in a module:
	    -add courses component in app.module.ts in declaration.
	3)Add an element in an HTML markup:
	   -create html markup app.component.html which contains html tag and use <course> which shows template element in page. 

3)

app.component.html:html combine component using add tag of component(template component)

///////////////////////////2021-01-29/////////////////////////////
1)Templates:
-binding view with logic.
-use {{variable}} to access from class to template .
-use for dynamic binding.
-also we can return variable using function and call in template. known as interpolation.

2)Directives:
-declare directive like ex.courses=["courses1","courses2","courses3"];
-In template,access directive . 
ex. <li *ngFor="let course of courses">
                {{course}}
    </li>
-ngFor is work like for loop

3)Services:
-In angular data coming from http end points.
-but we have no endpoint till now.
-so we need to make fake http endpoint.
-service is use to make fake http endpoint which return data. using function.
-name like ex.courses.service.ts

4)dependancy injection:
-dependancy injections means inject dependancy of component/class into its constructor.

-when use new keyword in constructor then it will be tightly couple with component.
-so solution is to instead of new ,use instanceName:serviceClassName in argument.
ex.constructor(service:CoursesService){
        // let service=new CoursesService();
        this.courses=service.getCourses();
    }
-this dependancy we need to register in app.module.ts(module) in provider.


-singleton:use only one instance for all component in module.

5)Generating Service Using Angular CLI:
-use command "ng g s serviceName"
