

1)get data from server:
-use http module
-Ex.,constructor(http:Http) {
      http.get('http://jsonplaceholder.typicode.com/posts')
      .subscribe(response=>{
        this.posts=response.json();
      });
   }

2)add data:
-get value using through arg in calling function from html
-use post for creating/add data.

-ex.,this.http.post(this.url,JSON.stringify(post))
        .subscribe(response=>{
          post['id']=response.json().id;
          this.posts.splice(0,0,post);
        })

3)update data:
-use patch or put method for update.
-get id from html for loop in argument.

-ex., updatePost(post){
     this.http.patch(this.url+'/'+post.id ,JSON.stringify({isRead:true}))
     .subscribe(response=>{
       console.log(response.json());
       
     });
      // this.http.put(this.url,JSON.stringify(post))
    }

4)delete data:
-use delete for delete data 
-get id from calling function argument from html
-ex.,deletePost(post){
      this.http.delete(this.url + '/' + post.id)
        .subscribe(response=>{
          let index = this.posts.indexOf(post);
          this.posts.splice(index,1);
        });
    }
