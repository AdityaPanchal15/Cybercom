6)pass data from child component to app.vue.
-it can possible using emmit ex.,
	
In App.vue:-
	
	<to-do-forms @todo-added="addToDo"></to-do-forms>
	
	<script>	
	export default {
	  methods: {
    		addToDo(toDoLabel) {
       			console.log(toDoLabel);
    		}
  	  },
	}
	</script>

In Child Component(ToDoForms.vue):-
	<script>
	export default {
	  data() {
	    return {
	      label: "",
	    };
	  },
	  methods: {
	    onSubmit() {
	      if (this.label === "") {
	        return;
	      }
	      this.$emit("todo-added", this.label);
	      this.label = "";
	    },
	  },
	};
	</script>
